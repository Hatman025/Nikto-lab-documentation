# Nikto-lab-documentation
Overview

Nikto is an open-source web server vulnerability scanner designed to identify potential security issues in web servers and web applications. It performs comprehensive checks against HTTP and HTTPS services to detect known vulnerabilities, outdated software versions, insecure configurations, dangerous files, and misconfigured server options.

This repository provides a structured, ethical, and educational approach to using Nikto for authorized security testing, learning, and defensive cybersecurity assessments.
